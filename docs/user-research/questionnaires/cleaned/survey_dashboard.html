方法论说明 ▶

2×2 分群矩阵定义

行维度：受访者当前是否处于减脂行动中（是→「行动中」，否→「非行动中」）。列维度：过去曾认真尝试减脂的次数（1-2次→「少次」，3次及以上→「多次」）。四象限命名：起步行动者 / 反复行动者 / 轻度脱轨 / 重度脱轨。

F 项分类标准

本次调研包含若干「频率选项题」（F 项），受访者选择频率描述词（如"经常""时不时""偶尔""很少"），汇总后以各频率段人数百分比呈现，并计算加权均值 q6_mean（4=经常 → 1=很少）。

Borda 评分说明

对于排序题或多选重要性题，采用 Borda Count 方法：将每个选项的排名位置转化为权重分数后加总平均。例如 4 选项题中，排第 1 得 3 分，排第 2 得 2 分，依此类推；未被选中计 0 分。最终分值反映该维度在样本中的相对重要性权重。

数据说明与排除标准

原始回收问卷共 91 份，排除 5 份（答题时间 < 90 秒或关键题项缺失），有效样本 N=86。CEIBS 渠道通过校友群投放，公开版通过社交媒体扩散。分群数据已对各子分群内部百分比独立归一化，与全样本「all」数据相互独立。联系方式留存率：CEIBS 37%（14/38），公开版 17%（8/48）。
