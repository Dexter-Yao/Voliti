# Go-To-Market 战略洞察

## 1. 当前定位的根本缺陷

「AI 减脂行为教练」作为品类描述仍需强化差异化。减脂 Agent 产品在市场上已有成百上千个，十几年前就开始出现类似方案。核心问题不是产品理念不好，而是**无法用一句话说清楚「为什么用户应该选你而不是其他任何一个」**。

展示文档中的差异化定位——「将 AI 与系统性行为改变方法论深度结合的现有产品数量 = 0」——站不住脚。Noom 的品牌叙事与此高度重合，任何竞品都可以声称自己做了行为科学整合。

## 2. 思考框架需要反转

错误路径：「我想做什么产品」→ 功能设计 → 寻找用户

正确路径：

1. **市场全景 Mapping**：国内外至少数十个竞品，了解产品形态、获客方式、盈利模式
2. **空间识别**：分析哪里有未被满足的需求，或者哪里的现有方案用户不满意
3. **盈利路径选择**：
   - (a) 竞争已有市场：从现有玩家手中争夺用户（需要明确的替代优势）
   - (b) 开辟新市场：找到别人没发现的空白点（需要验证需求真实性）
4. **反推产品方案**：基于以上商业判断来决定做什么、怎么做

## 3. 先卖后建

在产品开发之前先验证付费意愿。具体做法：

- 先找到 10 位愿意付费的用户
- 用**人力**而非 AI 或产品来服务他们（创始人亲自当教练）
- 观察哪些干预真的有效、哪些是假设
- 10 人 → 100 人后，再开始规模化和产品化

核心原则：用户付费购买的是**结果**（减脂成功、行为改变），不是技术手段（AI、Agent）。用什么方式交付，是你的问题而不是用户的问题。

## 4. 极度垂直的切入策略

「减脂」作为目标人群太宽泛。需要聚焦到**一句话能说清楚是谁**的具体群体。判断标准：

- 能在一个具体的地方找到他们（一个社群、一个平台、一个圈子）
- 他们有共同的生活场景和触发模式
- 他们之间会互相推荐

用户研究数据（[`07_User_Research.md`](07_User_Research.md)）显示：CEIBS 信任圈是高效种子渠道（留联系方式者 64% 为 CEIBS），核心客群为"反复行动者"（34%）和"重度脱轨"（19%），轻度脱轨不是优先目标。

## 5. 竞品深度调研要求

对市场上至少十到数十个国内外减脂相关 App / Agent 进行系统调研：

- 产品形态（工具 / 社群 / 教练 / 课程）
- 获客渠道与方式
- 盈利模式（订阅 / 单次付费 / 增值服务 / 广告）
- 目标人群与定价策略
- 用户评价中的不满与痛点

目标：形成完整的市场地图，识别价值攫取的具体机会点。

## 6. 已有资产的重新定位

用户研究（35 人访谈 + 跨平台数据 + 行为模式分析）是真实的、有深度的。问题不在于研究质量，而在于研究成果没有被翻译成商业语言：

- 已证明：痛点存在（知行分离、重启疲劳、情绪进食）
- 未证明：有人愿意为解决这个痛点付钱
- 未证明：付给你而不是付给别人

下一步需要回答的三个问题：

1. 谁会付钱（具体到能找到这个人）
2. 付多少钱（参照他们现在付给谁）
3. 为什么付给你（而不是现有替代方案）
