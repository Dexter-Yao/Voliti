# Go-To-Market 战略洞察

## 1. 定位与差异化

「AI 减脂行为教练」作为品类描述仍然不够。减脂产品在市场上数以百计，任何竞品都可以声称自己做了行为科学整合——Noom 的品牌叙事就是这么写的。

竞品研究（[09_Competitive_Landscape.md](09_Competitive_Landscape.md)）的核心判断是：行业里大多数产品在 Plan/Do 上最强（给方案、做记录），少数在 Check 上开始成熟（做趋势），而真正稀缺的是 **State（失控前识别）和 Act（写回长期模型）**。中国市场目前没有产品把"失控前预防 → 失控后 48-72 小时恢复 → 跨时间模式识别"做成主引擎。

用户研究（[07_User_Research.md](07_User_Research.md)）的需求侧数据与此收敛：目标用户中 43% 有效但反弹，51% 归因节奏被打乱，26% 归因方案难执行。核心需求不是"更好的计划"，而是"计划断裂后有人接住我"。

因此，Voliti 的差异化不应表述为"AI + 行为科学"（会被所有人声称），而应聚焦在供给侧最稀缺的能力上。但这个差异化能否转化为用户感知的价值，仍需要种子用户的真实使用验证。

## 2. 思考框架

导师反馈中指出的正确路径仍然成立：

1. **市场全景 Mapping** → 已完成。结论：直接竞品为 Noom/Lark/Omada/Wondr（美国行为骨架型），中国尚无直接竞品；最大结构性威胁来自通用 AI 对"问建议"入口的吞噬
2. **空间识别** → 白空间为"State-first 的私密复原系统"。在中国表现为"需求已在，品类未命名"
3. **盈利路径** → 属于开辟新品类而非争夺现有市场。需求真实性已有问卷支撑，但付费意愿尚未在产品层面验证
4. **反推产品方案** → 当前 S-PDCA 框架与竞品白空间高度吻合，无需调整核心架构

## 3. 先卖后建

这一原则不变：

- 先找到 10 位愿意付费的用户
- 用人力而非产品来服务他们（创始人亲自当教练）
- 观察哪些干预真的有效、哪些是假设
- 10 人 → 100 人后，再开始规模化

用户付费购买的是结果（行为一致性的提升），不是技术手段。

## 4. 目标人群

用户研究中同时满足「可触达」和「高意愿」的目标群体画像：

核心特征：25-40 岁，中基数（5-15 斤），有减脂经验但反弹过，多数是 AI 深度用户，MBTI 以 NT 型为主（直觉 + 思维），消费力强（58% 花过 ¥1,000+），在抖音和小红书上活跃。详细画像见 [07_User_Research.md](07_User_Research.md)。

**可以找到他们的地方**：抖音和小红书是主触达渠道（目标用户渗透率 83-89%），CEIBS 等信任圈是种子渠道，微信私域适合承接转化和留存。微信公众号/视频号在目标用户中渗透率仅 34%，不应作为主渠道。

## 5. 竞争环境

详见 [09_Competitive_Landscape.md](09_Competitive_Landscape.md)。关键判断：

**应避开的红海**：
- GLP-1 伴随行为支持层——WW/Omada/Lark/Noom/Hims/Ro 已在争夺，各自控制支付、医生、药物、品牌等关键资源
- AI 记录与计划生成层——Keep 的拍照记食物、ChatGPT 的饮食建议，这一层正在快速商品化
- "更温和的内容型减脂"——科学减脂内容已是主流意见领袖的基本功

**应关注的威胁**：
- Oura：已具备高频状态数据 + 用户信任 + 私密交互语境，如果把 weight-health 做成明确工作流，会非常接近 State Before Strategy
- ChatGPT：用户已经在里面问营养/运动问题，Memory 和 Apps 正在增强
- Keep/薄荷：体量和数据优势巨大，一旦把 AI 从"课程加速器"转向"行为脱轨预警与复原引擎"，跟进速度可能很快

## 6. 仍需验证的核心问题

用户研究和竞品分析已经回答了"痛点是否存在"（是）和"市场是否有空白"（是）。但以下三个问题仍然开放：

| 问题 | 当前状态 | 验证方式 |
|------|---------|---------|
| **谁会付钱** | 已识别高意愿 + 可触达的目标群体，消费力已验证（58% ¥1,000+）；但意愿不等于付费行为 | 种子用户 1v1 服务，观察付费转化 |
| **付多少** | 用户消费锚点：多数 ¥几百-3,000/年。竞品参考：Keep 会员 ¥218/年，Noom ~$199/年 | 定价测试 |
| **为什么付给 Voliti** | 竞品白空间已识别（State + Act），但用户是否感知到这个差异化尚未验证 | 种子用户是否在首次失控后回来继续使用 |
