# 用户研究综合报告

> 数据来源：Wave-1 问卷 87 人（CEIBS 37 + 公开 49）、Wave-2 问卷 92 人（公开渠道）、3 场深度访谈、小红书 3 人深度画像、Reddit 14 帖分析
>
> 详细数据：[Wave-1 分析](user-research/questionnaires/wave-1/wave-1_research_report.md)、[Wave-2 分析](user-research/questionnaires/wave-2/wave-2_analysis_report.md)、[定性研究](user-research/)

## 一、核心用户画像

Voliti 的目标用户是**反复尝试减脂但无法持续的人**——不缺知识，缺的是在真实生活压力下维持行为一致性的能力。

Wave-2 按「有效联系方式 × 意愿强度」分群后，**A 象限「反弹行动者」（35 人 / 38%）**是唯一同时满足可触达和高意愿的群体，画像如下：

| 维度 | 数据 |
|------|------|
| 年龄与性别 | 25-40 岁占 77%，男女均衡（49:51） |
| 体重基数 | 66% 中基数（5-15 斤），14% 大基数 |
| 居住 | 86% 与家人同住 |
| 减脂效果 | 43% 有效但反弹，26% 效果有限，23% 成功维持 |
| 失败归因 | 51% 节奏被打乱，26% 方案难执行（四象限最高） |
| 指导经历 | 51% 跟网上方法，26% 跟过专业人士 |
| 规划行为 | 40% 详细计划，43% 大致方向 |
| AI 深度 | 63% 聊过私事且觉得有用或经常依赖 |
| AI 信任 | 专业性(5.7) > 个性化(5.1) > 可执行(4.7)（Borda 排序） |
| 消费力 | 29% 花过 ¥3,000-10,000，29% 花过 ¥1,000-3,000 |
| 推送偏好 | 40% 每天 1-2 次，34% 关键时刻触发 |
| 产品形态 | App(3.5) > 小程序(3.0) > 网页（Borda） |
| 平台 | 抖音 89%，小红书 83% |
| MBTI | **94% N 型（直觉）、69% T 型（思维）**、56% I 型（内向） |

NT 型组合意味着目标用户天然偏好：理性分析自身行为模式、数据驱动的反馈、灵活而非刚性的方案。

**非目标用户**：D 象限「被动旁观者」（22 人）AI 接受度低（41% 只用 AI 聊工作）、外部指导经历极度匮乏（50% 从未获得有效帮助）、规划行为最弱（36% 直接行动不做计划）、MBTI 75% F 型——与目标用户在决策方式上有本质差异。

## 二、跨源验证的关键发现

以下发现经两轮问卷（共 179 人）与定性研究交叉验证：

**1. 知行差距是核心痛点。** Wave-1：62% 每周至少经历一次"知道不该但还是做了"，情绪/习惯是 #1 触发器（Borda 3.32）。Wave-2：53% 归因"节奏被打乱"，两题层次不同——前者是单次失控的触发器，后者是整体失败的归因叙事。访谈一致："道理我都知道。"

**2. 失控后的螺旋是行为中断的真正机制。** Wave-1：反复行动者 55% 的失控演变为当天放大或多天难以恢复。Wave-2 开放题中"一天的努力白费了""今天吃了明天减"的灾难化叙述反复出现，尤其集中在 B 象限（疲惫挣扎者）。

**3. 方案刚性是详细计划者的独有问题。** Wave-2 新发现：「方案难执行」仅出现在 A/C 象限（各 26-27%），B/D 象限为 0%。A/C 恰好是详细计划者最多的两个象限（40%/64%）。开放题佐证："减脂餐太难吃了""太极端太累就容易放弃"。越会做计划的人越觉得方案难执行——问题不在规划能力，在方案与生活的适配性。

**4. 79% 偏好私密方式（Wave-1），AI 接受度已跨过临界点（Wave-2）。** Wave-2 中仅 3% 没用过 AI，55% 已在私人话题中使用并觉得有用。AI 信任排序为专业性 > 个性化 > 记忆力，但已有专业指导背景的用户（C 象限，41% 跟过专业人士）是唯一把个性化排在专业性前面的组。

**5. 「想要专业指导但没条件」与低行动力高度相关。** Wave-2 中 Q5=D 共 12 人，全部落在 B/D 象限（低意愿组），无一人出现在 A/C。有能力获取专业指导的人已经获取了或不觉得需要。

## 三、对产品与 GTM 的含义

**教练设计**：
- Coach 首要能力是**节奏恢复**（失控后帮用户回来），不是计划制定。A 象限反弹率 43%，核心需求是断裂后的恢复路径
- 方案必须内置弹性（B 计划）。详细计划者归因方案难执行的比例是全样本的 1.6 倍
- Coach 默认人格应面向 NT 型设计：简洁、数据驱动、不过度共情。T/F 是四象限间最强的 MBTI 区分信号（活跃象限 61-80% T，被动象限 75% F）
- 信任建立路径：先展示专业性（新用户需求），再凸显个性化和记忆能力（深度用户需求）

**GTM**：
- 渠道：抖音（A 象限 89%）和小红书（83%）是主渠道；微信公众号仅 34%
- 推送节奏：默认每天 1-2 次；AI 深度用户对高频推送接受度是非深度用户的 2 倍
- 产品形态：App 是跨象限一致的首选形态
- 消费力：A 象限 58% 花过 ¥1,000+，与 Wave-1 的 Repeat Actors 一致

**分群方法论**：Wave-1 的行为态度变量聚类未找到自然分群。Wave-2 发现**行动信号**（是否留有效联系方式）× 意愿分的 2×2 可以产生画像差异显著的四象限——行动信号比态度变量更有分群力。

## 四、待验证假设

1. Q4「节奏被打乱」的具体构成——工作变动、社交干扰、情绪波动各占多少？结构化数据无法区分，需访谈
2. C 象限（意愿最高但不留联系方式）的转化条件——隐私顾虑、审慎态度还是不需要外部产品？
3. App vs Web 的真实优先级——产品形态排序明确（App 首选），Web MVP 能否完成种子验证？
4. B 象限（有联系方式但低意愿）是否可通过产品体验激活
