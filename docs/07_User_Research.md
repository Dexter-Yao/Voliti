# 用户研究综合报告

> 数据来源：1 场直接访谈（P01）、小红书 3 人深度 Profile 分析、Reddit 14 帖逐条分析、85 人问卷调查（CEIBS 37 + 公开版 49，排除无效 3 + 非目标 1）
>
> 详细数据：[用户研究目录](user-research/)（定性）、[问卷分析报告](user-research/questionnaires/cleaned/survey_analysis_report.md)（定量）

## 一、核心用户画像

Voliti 的目标用户是**反复尝试减脂但无法持续的人**——不是缺乏知识，而是缺乏在真实生活压力下维持行为一致性的能力。

问卷按 Q1（当前状态）x Q2（尝试次数）分为四个象限，其中**反复行动者**（34%）和**重度脱轨**（19%）合计 53%，是 Voliti 的核心客群。轻度脱轨（20%）的私密偏好率最低（59%）、社交偏好最高（29%）、留联系方式率最低（12%），更适合社交型产品，不是 Voliti 的优先目标。

反复行动者的典型特征：25-30 岁，工作节奏不规律（34% vs 全体 18%），每周至少失控 1-2 次，55% 的失控会演变为"当天放大"或"多天难以恢复"的螺旋。重度脱轨以 25-30 岁女性为主（75%），干扰总分最高，私教试过率最高（69%）且正面体验率达 91%，但当前已停止行动。

一位受访者的八个字精准概括了核心用户状态：**"从未停止，从未成功。"**

## 二、跨源验证的关键发现

以下五项发现经定性（访谈 + 社媒）与定量（问卷）三角验证，置信度高：

**1. 知行差距是核心痛点，不是知识不足。** 62% 的受访者每周至少经历一次"知道不该但还是做了"。P01 访谈原话："道理我都知道。"小红书用户："知道都知道，只是管不住。"Reddit 用户："I know what to do. Being consistent is the problem."问卷中"情绪或习惯"是所有象限的第一失控原因（Borda 均分 3.32/4），远超生理需求（2.71）和社交场合（2.49）。

**2. 工作压力是区分困难程度的最强信号。** 反复行动者（2.97/4）和重度脱轨（3.12/4）的工作压力干扰显著高于起步行动者（2.22/4）。开放题中失控场景高频出现"加班到很晚""出差应酬"。社交饮食虽然全体高影响率（56%），但跨象限差异不大——所有人都受社交饮食影响，但只有高压工作者被持续打垮。

**3. 失控后的螺旋是行为中断的真正机制。** 反复行动者中 55% 选择"当天放大"或"难以恢复"——一次失控不是单点事件，而是 2-3 天的连锁反应。受访者原话："本来想吃两粒，结果一发不可收拾。第二天又没忍住。第三天才恢复。"重度脱轨者描述了更严重的自责-报复循环：暴食 -> 断碳报复 -> 再暴食。Reddit 分析中识别的 All-or-Nothing 模式与此高度一致。

**4. 79% 偏好私密方式，但不等于想独行。** 重度脱轨中"希望有人支持但私密进行"占 50%（全体 27%）——他们不想独行，但也不想让别人看到自己的挣扎。最渴望的是一个只有自己知道的支持者。Reddit 高赞评论："Never tell anyone about a diet."

**5. 一对一教练有效但不可持续。** 私教一对一的正面体验率全场最高（81%），重度脱轨达 91%。但开放题揭示了核心局限——"教练一走，老习惯就占领高地。"AI 教练的价值在于：持续在场 + 私密性 + 可及性。

## 三、对产品与 GTM 的含义

**产品设计验证：**
- **State Before Strategy** 被数据直接支持——情绪/习惯是第一失控原因，Coach 应在感知情绪失控时立即介入。
- **LifeSign 预案** 应优先覆盖工作压力场景（加班、出差、高压工作期），而非笼统的"压力管理"。
- **失控后的 48-72 小时** 是关键干预窗口，Coach 需在此期间持续跟进，防止单次失控螺旋化。

**GTM 路径：**
- **CEIBS 是种子渠道**：留联系方式者中 64% 是 CEIBS，信任圈传播效率高。
- **反复行动者优先触达**——仍在行动、痛感明确、花费意愿最强（28% 花过 3000+）。重度脱轨留联系方式率最高（38%）但需不同的 onboarding 设计（当前不在行动中）。

**聚类分析结论：** 数据中不存在基于行为困难度的自然聚类（Gower + Average Linkage，Silhouette 均低于阈值）。用户是连续光谱，不是离散类型。产品应感知用户在光谱上的位置并动态调整，而非为不同"类型"设计不同路径。

## 四、待验证假设

1. 反复行动者是否愿意为 AI 教练付费（问卷只验证了痛感，未验证付费意愿）
2. 重度脱轨者是否能通过 AI 教练重新启动行动（当前无行为数据）
3. 第 6-8 周是否是关键衰减窗口（社媒观察支持，但问卷未覆盖时间维度）
4. 维持阶段的用户需求（Reddit 显示维持 > 减重的复杂度，但中国样本中尚无数据）
