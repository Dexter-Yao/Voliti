# Witness Card Visual System

All Witness Cards share the same brand frame. The image itself must not carry text.

## Constant Elements
- Illustration / ink / watercolor orientation rather than realism
- Paper texture
- Generous negative space
- Avoid real human faces
- Desaturated, gentle palettes
- No gamified elements

## Temperature Axis

### Warm Direction
Use for: growth, warmth, being seen, steady accumulation

Keywords:
- parchment background
- deep dark contrast
- watercolor brush texture
- quiet, personal, composed

### Cool Direction
Use for: breakthrough, strength, control, crossing a high-risk threshold

Keywords:
- warm dark background
- low-saturation cool blue accent
- ink-like linework
- calm, precise, stable

## Prohibited Elements
- Any text inside the image
- High saturation, explosive celebration, stars, badges, bursts
- Fitness-influencer aesthetics, realistic faces, generic inspirational posters
