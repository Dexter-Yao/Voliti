# Witness Card Positioning

A Witness Card is not a reward, not a motivational poster, and not an intervention tool.

It appears only after the user has already crossed a real threshold. Its job is to elevate a moment from "this happened" to "this was truly seen."

## Correct Mental Model
- Witnessing: recording a fact that has already become true
- Confirmation: helping the user notice that they have entered a new stage
- Keeping: preserving a relational artifact the user can revisit during low-motivation periods

## Misuse
- Using a card to push the user before they have actually done the thing
- Replacing support with a card while the user is still in distress
- Turning the card into a badge, points system, or celebration poster

## Product Boundary
- Intervention skills answer: "How do I get through this?"
- `witness-card` answers: "Why is this moment worth remembering?"
